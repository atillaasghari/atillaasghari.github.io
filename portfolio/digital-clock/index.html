    Microcontroller Clock Project

    This project is a simple digital clock implemented using a microcontroller, specifically an ATmega32. The clock features a 16x2 character LCD display to show the time in hours, minutes, and seconds. It also includes buttons for adjusting the hours and minutes.

    [Project github link]

    Hardware Components

    • ATmega32 microcontroller
    • 16x2 character LCD display
    • Push buttons for adjusting hours and minutes
    • Crystal oscillator (8MHz)

    Diagram and code


    Functionality

    The clock operates by incrementing the seconds, minutes, and hours counters. The time is displayed on the LCD screen in the format HH:MM:SS. Push buttons are used to adjust the hours and minutes when needed.

    Code Overview

    The code is written in AVR assembly language and utilizes the AVR Studio development environment. Here's a brief overview of the code:

    1. Initialization of hardware components and LCD configuration.
    2. Definition of custom characters for displaying clock symbols.
    3. Declaration of subroutines for updating hours and minutes.
    4. Main clock loop:
      • Continuous updating of the clock display.
      • Monitoring push buttons for hour and minute adjustments.

    Conclusion

    This microcontroller clock project demonstrates the basics of interfacing with external hardware components and implementing simple timekeeping functionalities. It serves as a foundation for more complex projects involving real-time clock features or additional functionalities.
